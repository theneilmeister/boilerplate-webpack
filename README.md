# boilerplate-webpack
A simple webpack boilerplate with hot loading and ES6


# Features

	- SCSS
	- ES6 (babel)
	- JSX


# Installation

	- git clone https://github.com/theneilmeister/boilerplate-webpack.git
	- npm install


# Usage

	- npm start
	- npm build
	- Then open a browser to http://localhost:8080
